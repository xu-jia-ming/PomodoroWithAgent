# PomodoroTable 需求说明（MVP / Windows 优先）

## 1. 项目目标
构建一个番茄钟桌面优先（Windows）的应用，采用前后端分离：
- 前端：Vue 3 + Vite + Element Plus
- 后端：FastAPI

当前阶段先完成 Windows Web 形态 MVP，后续可扩展 Android。

## 2. 功能范围（本期）
页面通过底部 Tab 按钮切换，包含 5 个模块：
1. 待办
2. 未来待办集
3. 锁机
4. 统计数据
5. 我的

## 3. 角色与使用场景
- 用户：个人学习/工作者
- 关键场景：
  - 管理待办与未来待办集
  - 启动番茄计时并记录完成情况
  - 在专注期启用锁机（MVP 先实现“逻辑开关 + 倒计时控制”，不做系统级强制）
  - 查看统计数据

## 4. 页面与交互需求

### 4.1 待办
- 展示待办列表（标题、预估番茄数、状态）
- 支持新增待办
- 支持完成/取消完成
- 提供番茄倒计时核心区（开始/暂停/重置、时长设置、完成记录）
- 新增待办后自动进入执行状态并启动番茄钟
- 列表支持“执行”操作，选中待办后立即进入番茄倒计时
- 支持修改、删除待办
- 已完成待办支持重复执行（可再次完成番茄）
- 支持正计时与倒计时模式切换
- 记录每个待办的累计完成次数（可重复完成）
- 番茄钟参数默认值：专注 25 分钟、短休息 5 分钟、长休息 15 分钟
- 每完成 4 个专注番茄钟自动进入 1 次长休息，其余进入短休息

### 4.2 未来待办集
- 展示未来待办集列表
- 支持新增未来待办集
- 支持查看未来待办集下关联待办数量（MVP 可直接展示 count）
- 支持修改、删除未来待办集

### 4.3 锁机
- 显示当前锁机状态（开启/关闭）
- 可设置专注时长（分钟）并启动
- 倒计时结束后自动解除

### 4.4 统计数据
- 今日完成番茄数
- 本周完成番茄数
- 专注总时长（分钟）
- 打断次数
- 待办完成率
- 最近专注会话明细（待办、时长、状态、结束时间）
- 展示历史番茄完成曲线图（基于历史完成时间）
- 保留历史完成时间列表，支持后续进一步分析

### 4.5 我的
- 显示用户基础信息（昵称、ID）
- 显示当前版本与后续占位设置项

### 4.6 底部 Tab
- 固定底部导航区域，点击切换对应页面
- 当前激活 Tab 高亮

### 4.7 设备尺寸 UI 切换
- 支持 Windows UI 与 Android 尺寸 UI 切换
- Android 尺寸 UI 以常见手机宽度进行容器适配，便于预览移动端布局

## 5. 非功能需求
- 代码分层清晰，前后端职责明确
- API 有统一错误返回结构
- 前端组件化，UI 使用 Element Plus
- 文档齐全（需求、架构、接口、开发流程）

## 6. 本期不做
- Android 打包与发布
- 系统级真实锁机（进程拦截、窗口屏蔽）
- 多用户认证体系（先用默认本地用户）

## 7. 验收标准（MVP）
- 可启动前后端服务
- 前端可看到 5 个 Tab 页面并可切换
- 待办/未来待办集可新增与展示
- 锁机页面可启动倒计时并结束
- 统计页展示后端返回的数据
- 文档位于 docs 目录且包含 git 规范流程
