# 开发与启动指南（Windows）

## 1. 环境要求
- Python 3.11+
- Conda（Miniconda 或 Anaconda）
- Node.js 20+

## 2. 后端启动（FastAPI + Conda）
在项目根目录执行：

```bash
cd backend
conda create -n pomodoro-table python=3.11 -y
conda activate pomodoro-table
pip install -r requirements.txt
uvicorn main:app --reload --host 127.0.0.1 --port 8000
```

后端健康检查：
- 打开 `http://127.0.0.1:8000/`

## 3. 前端启动（Vue3 + Element Plus）
另开一个终端，在项目根目录执行：

```bash
cd frontend
npm install
npm run dev
```

前端访问地址：
- `http://127.0.0.1:5173`

功能快速验证：
- 进入“待办”页，顶部“番茄倒计时”点击开始后应每秒递减
- 在“待办”页新增一个待办后，应自动开始该待办的番茄倒计时
- 倒计时结束后应提示完成，统计页“今日番茄/本周番茄/专注分钟”会更新
- 待办列表里“执行”可对已完成待办再次发起计时
- 计时模式可切换“倒计时/正计时”
- 页面顶部可切换 “Windows UI / Android UI” 预览

## 4. 常见问题
### 4.1 你之前执行了 `npm run serve` 失败
本项目基于 Vite，启动命令应为：

```bash
npm run dev
```

### 4.2 Conda 未生效
先初始化 shell：

```bash
conda init powershell
```

然后重开 PowerShell 再执行 `conda activate pomodoro-table`。

### 4.3 数据库和数据保存位置
- 当前使用 SQLite 本地数据库
- 数据文件路径：`backend/data/pomodoro.db`
- 后端重启后数据会保留；删除该数据库文件后数据会重置
- 当数据库为空时，系统会自动写入一批默认待办和最近一周历史会话数据，用于观察统计曲线
